# Dify Text2SQL Agent 详细项目计划

## 项目基础信息
- **项目名称**: Dify Text2SQL Agent
- **项目目标**: 构建可展示的技术作品，证明 BI+AI 开发能力
- **技术栈**: Python + MySQL + Vanna AI + Streamlit/FastAPI
- **开发周期**: 4周（2026-03-18 至 2026-04-14）
- **交付物**: 可运行Demo + 完整文档 + Docker部署方案

## Week 1: MVP 基础 (2026-03-18 至 2026-03-24)

### 目标
完成 SQLite Demo 的完整功能，验证核心架构可行性。

### 具体任务
| 任务ID | 任务描述 | 验收标准 | 优先级 |
|--------|----------|----------|--------|
| W1-01 | 初始化项目结构 | 标准Python项目目录结构 | P0 |
| W1-02 | 创建SQLite销售数据库 | 包含产品、订单、客户表，有示例数据 | P0 |
| W1-03 | 集成Vanna AI核心 | 能够基于schema生成SQL | P0 |
| W1-04 | 实现统一LLM适配器 | 支持Code Plan和普通API动态切换 | P0 |
| W1-05 | 开发Streamlit基础界面 | 输入框 + SQL显示 + 结果表格 | P0 |
| W1-06 | 实现fallback机制 | LLM失败时返回规则SQL或错误 | P1 |
| W1-07 | 编写Week1测试用例 | 覆盖核心功能路径 | P1 |

### 交付物
- 可运行的SQLite Demo
- 基础Streamlit界面
- LLM集成验证报告

### 风险控制
- **LLM API不可用**: 准备本地规则引擎作为备用
- **Vanna集成问题**: 使用最小可行集成方案
